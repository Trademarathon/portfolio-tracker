# Trade Marathon® — Installation Guide

## macOS (Apple Silicon M1/M2/M3)

### 1. Download

Get `Trade Marathon®_0.1.0_aarch64.dmg` from the release or share link.

### 2. Install

1. Double-click the DMG to open it.
2. Drag **Trade Marathon®** into the **Applications** folder.
3. Eject the DMG (right-click → Eject).

### 3. First Launch

macOS may show: *"Trade Marathon® cannot be opened because it is from an unidentified developer."*

**Fix:** Right-click the app → **Open** → **Open** again to confirm.

If you see *"damaged"* or *"can't be opened"*:  
**System Settings** → **Privacy & Security** → scroll down → click **Open Anyway** next to the blocked app.

### 4. Uninstall

Drag **Trade Marathon®** from Applications to Trash.

---

## System Requirements

- **macOS** 10.15 (Catalina) or later  
- **Architecture** Apple Silicon (aarch64)
